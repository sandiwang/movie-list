about this site

The purpose of the site is to demonstrate techniques of Angular

Implementations include $http, routing, facotry, animations, and directives.

The top movies list is a JSON file and is accessed through the $http service provided by Angular. The idea is to get the data from JSON file and display them in a clear and nice way to the users.

Also, in order to get detail information about each movie, I need an api to help with that. In this case, I am using OMDB API to get data from IMDB since IMDB doesn't seem to provide any official API. So the workflow is, when the page lands, get movie list from local file and display it. As user clicks for detail information, OMDB api will get what we want back so that we can again display them.

Anchor link/Smooth Scroll is done using jQuery method animate and is defined in a factory so that different controllers can share this function.

The go-to-previous-page function is implemented as a Back button shown in each movie's page. This is done in a directive named "backButton" which simply calls the $window function to go back to previous page.

If you are looking for the CSS mark-up framework of this site, it's Semantic-UI.

Some todos: (1) Migrate from ngRoute to UI Router (2) Scroll animations on viewport enter (3) Movie custom search function

All the photos and data are from IMDB. I do not own any rights.
